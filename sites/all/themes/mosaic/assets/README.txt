The assets folder contains media assets used by the Coral Nexus Mosaic theme.

Current folders:

- images: contains loose images used in css etc...
- icons: contains images that are compiled to an icon sprite.
- mosaic: contains images that are compiled to a mosaic sprite.
- vtiles: contains images that are compiled to a vertically tiling sprite.
- htiles: contains images that are compiled to a horizontally tiling sprite.

Add folders as needed.
